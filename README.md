# practical_midterm_1_ada# 🚀 Suite de Algoritmos HackerRank (Ejecución Local)

Este proyecto contiene soluciones optimizadas en Python 3 que leen automáticamente múltiples casos de prueba desde archivos de texto (`.txt`) sin necesidad de interactuar manualmente con la terminal.

---

## 📂 Estructura de Archivos
Asegúrate de mantener los archivos de código y texto juntos en la misma carpeta:

```text
├── sherlock_valid_string.py
├── test_sherlock.txt
├── lilys_homework.py
├── test_lilys.txt
├── stock_maximize.py
└── test_stock.txt
```
## Instrucciones de Ejecución
Abre la Terminal o Símbolo del Sistema (cmd).

Muévete a la carpeta del proyecto usando el comando cd:

```Bash
cd Ruta/De/Tu/Carpeta
Ejecuta los scripts uno por uno con Python:
```
```Bash
python sherlock_valid_string.py
python lilys_homework.py
python stock_maximize.py
```
